# 연못의 진실(v2) + 족자 설정 변경 + 히든 엔딩 2종 설계

날짜: 2026-07-10 / 상태: 승인됨

## 1. 연못 시스템 v2 — 고해 판정 폐지, 진실 카드 강제 공개
- 문제: 고해의 성립 여부를 플레이어(해월)가 판정하는 방식이 애매함
- 해결: 4페이즈부터 "연못 앞에 서겠다" 선언 → 자신의 [연못의 진실] 카드(봉인, 본인도 미열람)를 공개·낭독. 연못이 그 인물의 가장 감추고 싶은 진실을 대신 말하므로 판정 불필요, 거짓 개입 불가
- 달조각·계시 해금 규칙은 유지 (3개째 놓이는 순간 [연못의 계시] 공개)
- 신규 구성품: 연못의 진실 카드 5장 (인물별, 연못의 목소리 3인칭 서술)
- 미션 전환 카드 문구 정합화: "연못이 당신의 진실을 대신 말해줄 것입니다"

## 2. 서린 직업·소품 변경
- 민화가 → 동양화가
- 그림: 족자 + 세로 폭 동양화 (거북이와 소나무), 환명부는 족자와 그림 사이(배접 틈)에 은닉
- 연쇄 수정: 인물 목록, 서린 시트(나의 죄), 타임라인 18:00, 단서 '이안의 사진'·'하늘의 방 사진', 엔딩북 2곳, 연못의 진실 카드[윤서린]

## 3. 엔딩 4종 체제
분기 우선순위:
1. 타인 지목 1표 이상 → 결말 1 (단, 전원 타인 지목 + 달조각 0개 → 히든 [달이 삼킨 정원])
2. 전원 자기지목 → 결말 2 (단, 달조각 5개 → 히든 [수면 아래의 아이])

- [수면 아래의 아이] (히든 굿): 전원이 연못 앞에 서고 전원이 제 죄를 껴안았을 때만. 박이안이 수면 위로 등장해 작별 인사 — 게임 내내 목소리로만 존재하던 진짜 피해자의 유일한 등장. 형사의 연못 서기(빌린 목숨 인정)가 필수 조건이라 주제 완성
- [달이 삼킨 정원] (히든 배드): 아무도 연못 앞에 서지 않고 서로만 지목한 최악의 판. 진실 수장, 기억 소실. 마지막 문장("언젠가 다시, 달이 차오를 때까지")으로 리플레이 유도

## 4인 모드 검산
달조각 최대치: 플레이어 4인(서린·송이·해월·형사) + NPC 지석 카드5 자동 1개 = 5개 → 히든 굿엔딩 도달 가능
