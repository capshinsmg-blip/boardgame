# 연못 고해 시스템 + 행동지침 이원화 설계

날짜: 2026-07-10 / 상태: 승인됨

## 1. 연못 고해(告解) 시스템 — 조건부 엔딩 2 힌트
- 3페이즈부터 누구든 "연못 앞에 선다" 선언 후 자신의 죄를 자발적으로 고백 가능
- 성립 조건: 아무도 추궁하지 않은 자신의 죄를 스스로 밝혔을 때만. 단서로 몰린 뒤의 시인은 불인정
- 판정: 정해월 플레이어 (해월 본인의 고해는 형사가 판정)
- 보상: 달조각 토큰 1개를 보드판 연못에 배치 (최대 5개)
- 해금: 세 번째 달조각이 놓이는 순간 [연못의 계시] 카드 즉시 공개 — 자기지목=구원의 시적 암시
- 2개 이하면 계시는 끝까지 비공개 (엔딩 2 경로가 사실상 닫힘)
- 4인 플레이: NPC 지석 [카드 5] 자백 = 자동 달조각 1개
- 미션 전환 카드 4장에 연못 응답 안내 한 줄 추가 → 4페이즈부터 고해 동기 주입
- 신규 구성품: 달조각 토큰 5개, 연못의 계시 카드 1장

설계 의도: 고해는 은폐 미션을 스스로 깨는 행위 → "자신에게 불리한 행위의 자발적 선택"이 힌트 해금 조건이 됨. 시체가 잠든 연못이 진실을 받아 새벽을 여는 테마 일치.

## 2. 행동지침 이원화 (B안)
- 문제: 설정서의 페이즈별 지침 표가 자기 아크(후반 고백 전환)를 사전 스포 + 명령조라 RP 제약
- 해결: 페이즈별 지침 표 전면 삭제
  - 설정서: 페이즈 번호 없는 '연기 팁' 2~3줄 (팁 톤, 초반 스탠스만)
  - 후반 고백 유도: 이미 봉인된 미션 전환 카드가 전담
  - 형사: 페이즈 카드 공동 질문 + 기억 조각이 페이싱 담당, 팁 2줄만 유지

## 검증
- 지침 표(✦ 페이즈별 지침) 0회, 연기 팁 5블록 확인
- 고해 규칙·계시 카드·구성품 목록·전환 카드 문구 반영 확인
