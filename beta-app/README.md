# <달 아래 비밀 없는 정원> 베타플레이 앱

실물 구성품 없이 스마트폰만으로 게임을 진행하는 실시간 베타플레이 프로그램.
v2 원고의 모든 텍스트(설정서·단서 29장·연못의 진실·미션 전환·NPC 카드·엔딩 4종)가 탑재되어 있다.

## 실행 방법

1. `START_베타플레이.bat` 더블클릭 (최초 실행 시 패키지 자동 설치)
2. 진행 화면(`http://localhost:3000/host`)이 자동으로 열림 — 테이블 중앙 PC/태블릿에 띄워둘 것
3. 플레이어들은 진행 화면의 **QR을 스캔**해 각자 폰으로 접속 (같은 와이파이 필수)
4. 이름 입력 → 인물 선택 (4명이면 강지석 자동 NPC) → 호스트가 시간 모드 선택 후 [게임 시작]

* Windows 방화벽 창이 뜨면 "허용"을 눌러야 폰에서 접속된다.

## 게임 흐름 (자동 진행)

- **오프닝**: 각자 폰에서 자기 설정서(비밀 포함) 숙지 → 호스트가 1페이즈 시작
- **페이즈 1~5**: 서버 동기화 타이머. 형사 폰에 [다음 단서 공개] 버튼 — 누르면 전원 화면에 단서 등장
- **4페이즈**: 미션 전환 카드가 각자 폰에 비공개 도착. 연못 개방 — [연못 앞에 서겠다] 버튼 (형사는 5페이즈부터)
  - 연못의 진실 전체 공개 연출 + 달조각 카운트, 4인 모드에선 2번째 달조각에 NPC 지석 자동 참여, 3번째에 [연못의 계시] 자동 공개
- **타이머 종료** → "달이 기울었습니다" → **한 줄 문답**(전원 폰 입력, 전원 제출 시 동시 공개) → **형사의 기록** → 다음 페이즈 자동 전환
- **5페이즈 후**: (4인) NPC 카드5 → 형사의 재구성(기록지 5줄) → 최후의 발언(순서·타이머 자동) → **투표**(전원 폰 지목, 동시 공개)
- **엔딩 자동 판정**: 결말 1/2 + 히든 2종(달조각 0개/5개) → 엔딩·에필로그·타임라인 전원 공개

## 시간 모드
| 모드 | 페이즈 | 용도 |
|---|---|---|
| 표준 | 15/20/25/20/20분 | 정식 규칙 |
| 베타 단축 | 8/10/12/10/10분 | 첫 베타테스트 권장 |
| 기능 테스트 | 각 1분 | 동작 확인용 |

호스트는 언제든 [페이즈 조기 종료]로 시간을 앞당길 수 있다 (토론 소진 시).

## 검증
`node test/sim.js` — 4인+NPC 전체 플로우 자동 시뮬레이션 (29개 체크). 서버 실행 중에 돌릴 것.

## 구조
- `server.js` — 상태머신·타이머·판정 (모든 규칙은 서버 권위)
- `content.js` — v2 원고 전문. **원고 수정 시 이 파일만 갱신하면 된다**
- `public/` — 플레이어(모바일)·호스트 화면

## 온라인(클라우드) 버전
와이파이·PC 없이 각자 모바일 데이터로 접속하는 정식 배포판:
**https://moongarden.higgsfield.app**
- 첫 화면에서 [새 게임 방 만들기] → 진행 화면의 QR/참가 코드를 공유
- 진행 방식은 로컬판과 동일 (같은 게임 로직의 Cloudflare Durable Object 포팅)
- 검증: node test/cloud-sim.js wss://moongarden.higgsfield.app (24개 체크)
- 원고 수정 시 클라우드판 콘텐츠는 별도 저장소(moongarden)의 game-content.server.ts에 반영 필요
